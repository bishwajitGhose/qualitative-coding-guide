Essential Principles

1. Analytic Movement

You move UP (abstracting) and DOWN (checking against data) constantly. It's not a one-way journey.

2. Data ‚Üí Interpretation

Early stages stay close to data. Later stages require bold interpretation. Both are necessary.

3. Multiple Cycles

You'll code the same data multiple times, each time seeing something new. This is normal and productive.

4. Deliberate Reduction

From hundreds of data points to dozens of codes to a few categories to core themes. You're condensing meaning.
